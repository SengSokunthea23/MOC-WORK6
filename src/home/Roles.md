# III. Role

នេះគឺជា Role គឺជា «តួនាទី» <br>
អ្នកប្រើប្រាស់អាចធ្វើការកំណត់រាល់ `តួរនាទី` ទៅលើមុខងារមួយចំនួន :
- Read (អាចអាន)
- Write (អាចសរសេរ)
- Modify (អាចកែកែប្រែ)
- Remove (អាចដកចេញ)
- Action (អាចធ្វើសកម្មភាពផ្សេងៗបានទៀត)

នេះជាផ្ទាំងព័ត៌មានក្រោយពីអ្នកប្រើប្រាស់ចូលទៅកាន់គេហទំព័រ **Roles** រួចរាល់ :
